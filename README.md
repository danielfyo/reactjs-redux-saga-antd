## React Tutorial

ReactJS tutorial

### Tech stack:

- ReactJS

#### Steps

1. git clone
2. npm install
3. npm start

### fix chrome ssl error
4. in the chrome's address bar type chrome://net-internals/#hsts
5. Look for "Delete domain security policies" section
6. type in the textbox localhost
7. Press Delete btn
